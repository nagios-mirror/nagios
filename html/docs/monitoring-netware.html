Introduction

This document describes how you can monitor "private" services and attributes of Novell Netware servers, such as:

  • Memory usage
  • Processor utilization
  • Cache buffers
  • Active connections
  • Volume (disk) usage
  • etc.

Publicly available services that are provided by Netware servers (HTTP, FTP, etc.) can be monitored easily by following the documentation on monitoring publicly available services.
